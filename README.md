# Aether Hub

A small universe of hand-built pages, zero frameworks:

- **The Universe** · 140,000 WebGL particles morphing between five worlds. Right-drag to orbit, scroll to zoom, left-click a feature to inspect it, Space for the next galaxy.
- **Warframe, an introduction** · a scrolling primer for new Tenno: factions, missions, story timeline, Railjack, and why it is actually free.
- **For Wika** · a question with one right answer.

## Structure

```
index.html          landing page
/html               pages (hub + worlds)
/css                one stylesheet per page + common.css
/js                 one script per page
/images             screenshots (web-optimized)
```

Everything is plain HTML/CSS/JS. Open `index.html` or serve the folder statically.
